What is it?

A Timer to keep Daily Scrum meetings short.

A lot of developers do not like to spend too much time on Scrum "ceremonies". This tool is meant to help you get a viual indication of time spent to avoid it drawing out for online daily Scrum meetings.

How to use

  1. Go to Participants and make sure that everyone you want in the Scrum Timer
  2. Go to the Timer and mark anyone who is not present as absent.
  3. click the start button to start the timer
  4. Click the next button for each subsequent participant, and finally the stop button

Notes

  • If Max time is selected, the time left is distrubuted among the rest of the people in the meeting.
  • If the version badge below this list is older than the version of the newest release press CTRL + F5 to refresh the page.
